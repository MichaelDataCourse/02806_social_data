Data Part 3

Each row is a vegetable or fruit.
Data has columns:
name,frost,jan,feb,mar,apr,may,jun,jul,aug,sep,oct,nov,dec

* 'name': Name of the fruit/vegetable
* 'frost': can take values 0 or 1:
    * 1 if Harvest and availablity ends with frost, as early as September or October.
    * 0 otherwise
* months can take 0, 1 or 2 as values:
    * 0 if fruit/vegetable is not available that month
    * 1 if fruit/vegetable is harvested during that month (i.e. fresh).
    * 2 if fruit/vegetable is only available from storage during that month.
